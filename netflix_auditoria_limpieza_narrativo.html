Análisis narrativo de Netflix: revisión, auditoría y limpieza de datos¶

Portafolio

Este notebook trabaja con el dataset Netflix Titles y sigue un flujo parecido al notebook del Titanic, pero enfocado en auditoría, limpieza y análisis exploratorio.

El objetivo no es crear un modelo predictivo, sino practicar un proceso realista de datos:

  • cargar y entender el archivo CSV,
  • revisar columnas, tipos y valores faltantes,
  • auditar problemas de calidad,
  • limpiar datos de forma documentada,
  • crear variables nuevas útiles para análisis,
  • visualizar patrones principales,
  • guardar una versión limpia del dataset.

La idea es que cada paso explique qué se hace y por qué se hace.

1. Carga de datos¶

Cada fila representa un título disponible en Netflix. Las columnas principales son:

  • show_id: identificador del título.
  • type: tipo de contenido, puede ser Movie o TV Show.
  • title: nombre del contenido.
  • director: director o directores.
  • cast: reparto principal.
  • country: país o países relacionados con la producción.
  • date_added: fecha en que el contenido fue agregado a Netflix.
  • release_year: año de estreno.
  • rating: clasificación por edad o audiencia.
  • duration: duración en minutos para películas o temporadas para series.
  • listed_in: categorías o géneros.
  • description: descripción del contenido.

El archivo esperado se llama netflix_titles.csv. El código busca el CSV en varias ubicaciones comunes para que sea más fácil correr el notebook.

3. Auditoría de calidad¶

La auditoría revisa problemas que pueden afectar el análisis:

  1. Duplicados: filas completas repetidas o identificadores repetidos.
  2. Valores faltantes: columnas con nulos importantes.
  3. Valores fuera de rango: años imposibles o duraciones mal registradas.
  4. Consistencia: columnas donde el contenido no corresponde con su significado.
  5. Cardinalidad: número de valores únicos por columna.

En este dataset hay un caso importante: algunas filas tienen la duración dentro de rating, dejando duration vacío. Eso se corrige en la limpieza.

4. Limpieza de datos¶

Vamos a limpiar una copia del dataset para conservar el original intacto.

Decisiones de limpieza¶

  1. Duplicados: se eliminan filas completamente repetidas si existen.
  2. Texto: se quitan espacios extra en columnas de tipo texto.
  3. duration: si la duración quedó por error en rating, se mueve a duration.
  4. rating: los valores faltantes se rellenan con Sin clasificación.
  5. director, cast y country: se reemplazan nulos por etiquetas descriptivas como No especificado.
  6. date_added: se convierte a fecha real con pd.to_datetime.
  7. duration: se separa en número (duration_value) y unidad (duration_unit).
  8. Tipos de contenido: se valida que type solo tenga Movie o TV Show.

Estas decisiones evitan borrar demasiada información y dejan el dataset listo para análisis.

In [5]:
print("--- Iniciando limpieza de datos ---")

df = df_original.copy()
filas_inicio = len(df)

# 1) Eliminar duplicados exactos.
df = df.drop_duplicates().copy()

# 2) Quitar espacios extra en columnas de texto.
text_cols = df.select_dtypes(include="object").columns
for col in text_cols:
    df[col] = df[col].astype("string").str.strip()

# 3) Corregir filas donde la duración quedó en rating.
mask_duration_in_rating = (
    df["duration"].isna()
    & df["rating"].astype("string").str.contains(r"min|Season", case=False, na=False)
)
df.loc[mask_duration_in_rating, "duration"] = df.loc[mask_duration_in_rating, "rating"]
df.loc[mask_duration_in_rating, "rating"] = pd.NA

# 4) Rellenar faltantes categóricos con etiquetas claras.
df["director"] = df["director"].fillna("No especificado")
df["cast"] = df["cast"].fillna("No especificado")
df["country"] = df["country"].fillna("No especificado")
df["rating"] = df["rating"].fillna("Sin clasificación")
df["duration"] = df["duration"].fillna("No especificado")

# 5) Convertir date_added a fecha real.
df["date_added"] = pd.to_datetime(df["date_added"], errors="coerce")

# 6) Extraer número y unidad de duración.
duration_extract = df["duration"].astype("string").str.extract(r"(\d+)\s*(.*)")
df["duration_value"] = pd.to_numeric(duration_extract[0], errors="coerce")
df["duration_unit"] = duration_extract[1].replace("", pd.NA)

# 7) Validar tipo de contenido.
df = df[df["type"].isin(["Movie", "TV Show"])].copy()

filas_final = len(df)

print("Filas iniciales:", filas_inicio)
print("Filas finales:", filas_final)
print("Filas eliminadas:", filas_inicio - filas_final)

print("\nValores faltantes después de la limpieza:")
display(df.isna().sum().sort_values(ascending=False).to_frame("faltantes"))

5. Creación de variables nuevas¶

Muchas veces los datos útiles no vienen listos en una sola columna. Aquí creamos variables para analizar mejor el catálogo.

Crearemos:

  • added_year: año en que el contenido fue agregado a Netflix.
  • added_month: mes en que fue agregado.
  • main_country: primer país listado en country.
  • main_genre: primer género listado en listed_in.
  • num_genres: cantidad de categorías asignadas al título.
  • num_cast: cantidad aproximada de personas listadas en el reparto.
  • is_movie: indicador de película.
  • is_tv_show: indicador de serie.
  • movie_minutes: duración en minutos solo para películas.
  • tv_seasons: número de temporadas solo para series.
  • years_between_release_and_added: diferencia entre año de estreno y año agregado a Netflix.

# Año y mes de agregación a Netflix.
df["added_year"] = df["date_added"].dt.year
df["added_month"] = df["date_added"].dt.month

# País principal: primer país de la lista.
df["main_country"] = df["country"].str.split(",").str[0].str.strip()

# Género principal: primer género de la lista.
df["main_genre"] = df["listed_in"].str.split(",").str[0].str.strip()

# Número de géneros por título.
df["num_genres"] = df["listed_in"].str.split(",").apply(lambda x: len(x) if isinstance(x, list) else 0)

# Número aproximado de actores o actrices listados.
df["num_cast"] = df["cast"].apply(
    lambda value: 0 if value == "No especificado" else len(str(value).split(","))
)

# Indicadores por tipo.
df["is_movie"] = (df["type"] == "Movie").astype(int)
df["is_tv_show"] = (df["type"] == "TV Show").astype(int)

# Separar duración de películas y series.
df["movie_minutes"] = np.where(df["type"] == "Movie", df["duration_value"], np.nan)
df["tv_seasons"] = np.where(df["type"] == "TV Show", df["duration_value"], np.nan)

# Diferencia entre estreno y llegada a Netflix.
df["years_between_release_and_added"] = df["added_year"] - df["release_year"]
df.loc[df["years_between_release_and_added"] < 0, "years_between_release_and_added"] = np.nan

print("Nuevas columnas creadas:")
new_cols = [
    "added_year", "added_month", "main_country", "main_genre", "num_genres",
    "num_cast", "is_movie", "is_tv_show", "movie_minutes", "tv_seasons",
    "years_between_release_and_added"
]
print(new_cols)

display(df[new_cols].head())

10. Conclusiones¶

Después de la auditoría y limpieza se puede concluir que:

  1. El dataset original tenía valores faltantes principalmente en director, cast, country, date_added, rating y duration.
  2. No todos los faltantes debían eliminarse; en columnas como director, cast y country era mejor usar una etiqueta como No especificado.
  3. La columna date_added necesitaba convertirse a formato de fecha para poder analizar años y meses.
  4. La columna duration contenía dos tipos de información: minutos para películas y temporadas para series. Por eso se separó en variables nuevas.
  5. Se crearon columnas como main_country, main_genre, movie_minutes y tv_seasons, que permiten hacer análisis más claros.
  6. La mayoría del catálogo está compuesto por películas, aunque también hay una cantidad importante de series.
  7. Estados Unidos, India y Reino Unido suelen aparecer entre los países principales con más títulos.
  8. Los géneros más frecuentes permiten entender mejor el enfoque del catálogo.

Este flujo muestra una parte esencial del trabajo con datos: antes de analizar o graficar, primero hay que revisar, auditar y limpiar.
